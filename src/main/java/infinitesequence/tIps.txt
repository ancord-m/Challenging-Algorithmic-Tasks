Необходимо понимать, что последовательность на входе может быть
образована следующим образом:
1. Число действительно уникально (199, 1947293, 299, 284)
2. несколько нулей подряд
3. числа на стыке изменения старшего разряда или перехода в еще больший разряд (99100, (4)8495(0) )
4. есть несколько последовательных чисел без огрызков (274275276)
5. есть несколько последовательных чисел огрызанных ( 2742752762(77), (2)742752762(77), (27)4275276277 )
6. одно уникальное число с огрызками от соседей побокам ( (947)194729(473), (947)1947294(73)  )
7. огрызки от соседей (10)3333410(33335), т.е 1033334, 1033335; (47)19.472(0); (47)08.470(9)
8. взяты цифры из любого числа, но это уже самостоятельное число либо один из случаев выше и встретится раньше
9. Прорабоать вариант с уменьшением старшего или увеличением младшего разряда
Типа младшая 9 и ее увелиечние повлечет изменение в десятичном разряде. Или же самый старший разряд надо уменьшить.


1. Есть некое число. Из него выделены лишь некоторые цифры и они есть входные данные.
Теперь остается лишь делить его пополам в разных местах и комбинировать из этого последовательные числа
Например, берем 33334218, отсюда выделяем 34218. Все. Оно само по себе.
- 34218 - само себе число
- (4218)3.4218(4)
- (218)34.218(35)
- (18)342.18(343) - находим первое вхождение 18342, плюсуем 2 и ответ готов
- (8)3421.8(3422)
Оно может не дать верного ответа, например, 1947294, оно образовано из (947)1.9472.94(73)
а 9471 встретиться намного раньше, чем 1947294
